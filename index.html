Bootstrap é o framework HTML, CSS e JS mais popular para desenvolver site responsivos, com prioridade para dispositivos móveis

Download Bootstrap

Versão atual 3.3.1

Criado para todos, em todo lugar.

Bootstrap facilita e agiliza o desenvolvimento de sites. Feito para pessoas com diferentes níveis de habilidades, dispositivos de todos os formatos, e projetos de qualquer tamanho.

Pré-processadores

O Bootstrap trabalha com Vanilla CSS, mas o código utiliza os dois pré-processadores CSS mais populares, Less e Sass. Comece rapidamente com o CSS já compilado ou crie uma versão personalizada.

Um framework, todos os dispositivos.

O Bootstrap possibilita, de forma fácil e eficiente, que seus sites e aplicativos funcionem em todos os dispositivos, de celulares a tablets e desktops utilizando CSS media queries.

Cheio de recursos

Com Bootstrap, você tem uma bela e extensiva documentação em português com elementos HTML comuns, dúzias de componentes HTML and CSS personalizádos, e plugins jQuery incríveis.

Bootstrap tem código aberto. Está hospedado, mantido e em desenvolvimento no GitHub.
